# Angular Mandatory Exercises — DN Java FSE

All 10 Angular hands-on exercises completed.

## How to Run Any Exercise

```bash
# 1. Install Angular CLI (one time)
npm install -g @angular/cli

# 2. Create a new Angular project
ng new my-app --no-standalone
cd my-app

# 3. Replace src/app/ with the exercise folder's src/app/ contents
# 4. Start the dev server
ng serve

# 5. Open http://localhost:4200
```
